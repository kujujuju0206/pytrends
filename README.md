# キーワードボリューム検索ツール

このツールは、Google、Instagram、TikTokの指定キーワードに対する期間別の検索ボリューム増減推移を表示するWebアプリケーションです。PyScriptを使用してPythonの処理をブラウザ側で実行します。

## 機能

- 複数のキーワードを同時に検索（カンマ区切りで入力）
- Google、Instagram、TikTokの3つのプラットフォームから選択可能
- 過去3ヶ月、6ヶ月、12ヶ月の期間を選択可能
- 検索ボリュームの推移をグラフで表示
- 各プラットフォームを異なる色で識別（Google: 青、Instagram: 紫、TikTok: 黒）

## 技術スタック

- HTML/CSS: UIの構築
- PyScript: ブラウザ上でのPython実行
- Plotly: グラフ描画
- Keywordtool.io API: 検索ボリュームデータの取得

## 使い方

1. `index.html`をWebブラウザで開きます
2. キーワード入力欄に検索したいキーワードをカンマ区切りで入力します（例: ファッション, 旅行, スポーツ）
3. プラットフォームを選択します（複数選択可能）
4. 期間を選択します
5. 「検索」ボタンをクリックします
6. グラフが表示されるまで待ちます

## APIについて

このアプリケーションは[Keywordtool.io API](https://keywordtool.io/api)を使用しています。APIキーは実際の運用では安全に管理する必要があります。

## デモ機能

APIが利用できない環境でテストするために、デモデータ生成機能が実装されています。コード内の`test_with_demo_data()`関数のコメントを外すことで、APIを使わずにデモデータでグラフを表示できます。

## 注意事項

- APIキーは実際の運用では環境変数などから安全に取得する方法を検討してください
- 大量のキーワードを一度に検索すると、APIのレート制限に達する可能性があります
- ブラウザのCORS制限により、一部の環境ではAPIリクエストが失敗する場合があります
